# Customer Segmentation & Prediction

## Project Overview
This project focuses on segmenting customers using clustering algorithms and building machine learning models to predict customer churn. The goal is to help businesses make data-driven decisions.

## Dataset
The dataset includes customer tenure, charges, contract details, payment methods, billing preferences, and churn status.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
