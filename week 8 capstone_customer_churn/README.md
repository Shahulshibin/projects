# Customer Churn Capstone Project

## Overview
This project analyzes customer churn data to identify churn drivers and recommend retention strategies.

## Dataset
500+ customer records with churn labels.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Scipy

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis
- Hypothesis Testing
- Customer Segmentation

## Key Insights
- Monthly charges strongly impact churn
- Contract length reduces churn
